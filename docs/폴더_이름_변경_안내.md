# 폴더 이름 변경 안내

## 현재 상태

- **공식 명칭**: PAIOR(파이오르)
- **프로젝트 폴더**: `PAIOR_Project` (workspace 경로)
- **소스 파일**: PAIOR가 포함된 파일명 없음 (모두 정상)
- **import 경로**: PAIOR를 참조하는 경로 없음 (모두 상대 경로)

## 폴더 이름을 PAOIR_Project로 변경하려면

Windows 탐색기 또는 터미널에서 **수동**으로 폴더 이름을 변경하세요.

### Windows 탐색기

1. `e:\PAIOR_Project` 폴더의 **부모 폴더**로 이동
2. `PAIOR_Project` 폴더 우클릭 → **이름 바꾸기**
3. `PAOIR_Project`로 입력 후 Enter

### PowerShell

```powershell
Rename-Item -Path "e:\PAIOR_Project" -NewName "PAOIR_Project"
```

### 주의사항

- Cursor/VS Code를 **종료한 상태**에서 변경하세요.
- 변경 후 `e:\PAOIR_Project` 경로로 프로젝트를 다시 열어주세요.
- README.md, PREVIEW.md의 경로는 이미 `PAOIR_Project`로 되어 있습니다.
