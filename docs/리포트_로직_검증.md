# PAIOR 리포트 로직 검증

## 동점 처리 우선순위 (Tie-breaking)

**규칙**: I > A > R > O

| 순위 | 요인 | 코드 |
|------|------|------|
| 1 | 탐구(I) | 0 |
| 2 | 추진력(A) | 1 |
| 3 | 관계(R) | 2 |
| 4 | 질서(O) | 3 |

**구현 위치**: `src/lib/analysisEngine.js`  
- `TIE_BREAK_PRIORITY = { I: 0, A: 1, R: 2, O: 3 }`
- `compareAOIR()`: 동점 시 우선순위값으로 정렬

## 점수 등급 기준 (50점 만점)

| 등급 | 점수 범위 | 상태 |
|------|-----------|------|
| 높음 | 40점 이상 | 핵심 강점/활성 상태 |
| 보통 | 25~39점 | 안정적/유연한 상태 |
| 낮음 | 24점 이하 | 잠재 영역/미활성 상태 |

**구현 위치**: `src/lib/analysisEngine.js`  
- `getScoreGrade(score)`: 40+ → 높음, 25~39 → 보통, ≤24 → 낮음

**적용**: AOIR 차트, P 차트, 리포트 점수 표시, `getHighPDynamics` (40+), `getTopPDynamics` (25+)
