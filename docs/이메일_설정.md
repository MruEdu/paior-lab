# PAIOR 이메일 전송 설정

진단 결과를 PDF로 이메일 전송하려면 SMTP 설정이 필요합니다.

## Gmail 사용 시

1. **Google 계정** → 보안 → 2단계 인증 **활성화**
2. **앱 비밀번호** 생성: 보안 → 앱 비밀번호 → 메일 선택 → 기기 선택 → 생성
3. 생성된 16자리 비밀번호를 복사

## .env 파일 설정

`.env` 파일에 다음을 추가하세요:

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=귀하의이메일@gmail.com
SMTP_PASS=앱비밀번호(16자리)
SMTP_FROM=귀하의이메일@gmail.com
```

## 다른 메일 서비스

| 서비스 | SMTP_HOST | SMTP_PORT |
|--------|-----------|-----------|
| Naver | smtp.naver.com | 587 |
| Daum | smtp.daum.net | 465 |
| Outlook | smtp.office365.com | 587 |

각 서비스의 앱 비밀번호 또는 전용 비밀번호를 사용하세요.
